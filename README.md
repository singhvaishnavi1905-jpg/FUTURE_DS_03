# 🏦 Bank Marketing Campaign Analysis

## 📌 Objective
Analyze a bank telemarketing campaign to identify factors influencing client subscription to term deposits.


## 📊 Dataset
- Source: UCI Machine Learning Repository
- Records: 41,176 | Columns: 21
- Target Variable: y (yes/no subscription)


## 🔍 Key Findings
- Overall conversion rate: 11.27%
- May had highest calls but lowest conversion
- Longer call duration strongly linked to subscription
- Retired and student segments showed better conversion


## 🛠️ Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI
- Microsoft Word

## 📈 Dashboard Preview
![Power BI Dashboard](dashboard.png)

## 💡 Business Recommendations
1. Focus campaigns in March, September and October for better conversion rates
2. Target retired and student client segments more specifically
3. Train agents to improve call quality and engagement duration
4. Avoid mass calling in May without better targeting strategy
5. Use economic indicators (euribor3m, emp.var.rate) to time campaigns better
